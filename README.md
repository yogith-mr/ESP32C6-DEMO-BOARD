# 🚀 ESP32-C6 Development Board

## 📌 Overview
This project showcases the complete design and development of a custom **ESP32-C6 Development Board**, including schematic capture and PCB layout using KiCad. The board is designed for embedded systems and IoT applications, focusing on reliable hardware design, signal integrity, and efficient routing.

---

## 🎯 Objectives
- Design a custom ESP32-C6 based hardware platform  
- Implement robust power distribution and signal routing  
- Develop a compact and optimized PCB layout  
- Prepare the design for fabrication and real-world deployment  

---

## 🛠️ Design Details

### 🔹 Schematic Design
- Designed using **KiCad**
- Includes ESP32-C6 module, power supply circuitry, and essential interfaces  
- Proper decoupling capacitors and component selection applied  

### 🔹 PCB Layout
- Multi-layer PCB design  
- Optimized component placement for performance and compactness  
- Ground planes and power routing carefully implemented  

### 🔹 Routing
- Clean and structured trace routing  
- Signal integrity considerations applied  
- EMI-aware design practices followed
